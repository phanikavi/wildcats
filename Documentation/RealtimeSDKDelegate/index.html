on​Participant​Joined(room:​participant:​) 

func onParticipantJoined(room: Room, participant: RemoteParticipant)

Raised when a remote participant joined the room. Even though the participant has successfully joined, its media stream is not yet being stream. The event {@link #remotestream} will indicate that.

example

room.on('participantJoined', p => {
    const div = document.createElement('div');
    // Let the SDK create the video tag and update the stream when needed
    participant.attach(div);
    document.body.appendChild(div);
});

on​Local​Stream​Updated(room:​participant:​) 

func onLocalStreamUpdated(room: Room, participant: LocalParticipant)

Raised when local stream is added, removed or changed. This event is handled internally when the SDK was used to create the video element via {@link LocalParticipant#attach}. If that is not the case then the application is responsible to update the video element's srcObject

example

room.on('localStream', p => {
    const el = document.getElementById('localVideoEl');
    el.srcObject = p.mediaStream;
});
