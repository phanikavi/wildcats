public func setVideoView(_ view: UIView)

  • Attach video element to provided DOM element. mediaStream is automatically

  • updated. No need for the application to listen for localStream or remoteStream

  • events and manually update the video's srcObject.

  • @param {object} el - DOM element to attach the video to.

  • @param {object} [style] - CSS styles to add to video element.

  • @example

  • const room = await joinRoom();

  • room.localParticipant.attach(document.getElementById('localVideo'), {

  • width: '300px',

  • 'object-fit': 'cover'

  • });
